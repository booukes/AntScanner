# 📄 AntScanner

**AntScanner** is a rugged, luxury document scanner built for ultimate simplicity and elegant UI. Designed as a dedicated tool to eliminate clutter and ads, it offers a pure, high-end scanning experience.

## ✨ Premium Features
- **100% Free & No Ads:** No subscriptions, no hidden payments, and absolutely zero advertisements. A pure experience for a pure purpose.
- **ML Kit System Scanning:** Leveraging Google's latest AI algorithms for automatic edge detection and document cleanup.
- **Interface:** Full Material 3 compliance, dynamic colors, smooth list entry animations.
- **Thumbnail Previews:** Intelligent real-time rendering of PDF first pages directly on the list using the Coil library.
- **Privacy First:** Minimal permissions. Files are saved directly to the system `Downloads` folder for easy access and security.


## 🛡️ Philosophy
AntScanner was built with a simple philosophy. Most scanners on the market are bloated with subscriptions and tracking. This app is, and will always remain:
- **No In-App Purchases.**
- **No Data Collection.**
- **No Account Required.**
- **No Ads. Never.**

## 🚀 Installation
Download the latest APK from the [Releases](https://github.com/boouke/AntScanner/releases) section and install it on your Android device (Android 11+ required). App also available on FDroid

## 🛠️ Technical Stack
- **Language:** Kotlin
- **UI:** Jetpack Compose (Material 3)
- **Database:** Room Persistence
- **Scanning Engine:** Google ML Kit Document Scanner
- **Image Loading:** Coil with a custom PDF Decoder
